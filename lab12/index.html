Lab 12 - Conditionals

Challenge

The challenge of this lab was to create a function that depends on conditionals.

Problems

My main problem for this lab was getting retrieving the value of the input. I found out that my issue was that I assigned the value of the input to a variable before the event listener for the button, causing nothing to be assigned to the variable. My other issue was that I wasn't sure how to assign the value of the input -- after some searching on w3schools, I found out that it was because I was using "$("#input").value", however with jQuery, you need to use "$("#input").val()" instead.

Reflection

Aside from the two problems I listed above, the rest of the lab went rather smoothly. Conditionals in JavaScript weren't hard for me to understand/learn due to my prior experience with Python, so this lab wasn't too difficult.
